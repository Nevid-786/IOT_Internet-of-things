# 🌐 IOT_Internet-of-things

This repository contains multiple **IoT (Internet of Things) projects** combining **ESP32, Python, and Computer Vision**.  
Each project is structured with clear documentation, source code, and setup instructions.

---

## 📂 Repository Structure

- **Communication_protocols/**
  - ESP-NOW (Two-Way Communication) examples
- **ESP32_microcontroller_with_WebCam/**
  - Control ESP32 components using laptop/webcam face detection

---

## ⚙️ Features

- 🔗 **ESP32 Communication Protocols** (ESP-NOW, Serial, etc.)
- 📷 **Computer Vision Integration** with OpenCV & Mediapipe
- 🚦 **Driver State Monitoring System**
- 💡 **ESP32 Peripheral Control** (LEDs, sensors, etc.)
- 📡 IoT-based communication between devices

---

## 🛠️ Requirements

- **Arduino IDE** (with ESP32 boards installed)
- **Python 3.8 – 3.11**  
  - Libraries: `opencv-python`, `mediapipe`, `numpy`, `pyserial`, `absl-py`
- **ESP32 Dev Board**
